# 技術スタック

## フロントエンド

- **HTML5**
- **Tailwind CSS**（CDN経由）
- **Alpine.js**（CDN経由）
- **JavaScript (ES6+)**

## バックエンド

- **PHP 8.0+**
- **MySQL 8.0+**
- **REST API**（JSON形式）

## 開発環境

- Visual Studio Code
- Git
- 将来的にLaravel移行も検討

## 制約事項

- レンタルサーバーでの動作を前提
- CDN経由でのライブラリ利用（npm等の依存関係管理なし）
- フェーズ1ではサーバーサイド開発なし（ローカルストレージ中心）
